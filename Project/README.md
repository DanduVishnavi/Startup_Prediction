## 🚀 Startup Success Predictor

AI Startup Success Predictor is a Machine Learning-based web application 
that predicts whether a startup will operate successfully, fail, or get acquired.  

The system analyzes financial strength, investor activity, business milestones, 
and market recognition to generate a prediction along with confidence score 
and risk level.  

This project demonstrates end-to-end integration of a trained ML model 
with a Flask web application and a modern interactive user interface.
